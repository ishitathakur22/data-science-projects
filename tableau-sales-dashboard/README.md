# 📊 Tableau Sales Dashboard

An interactive sales analytics dashboard built in Tableau.

## What it shows
- 💰 Total Sales Value — $1,167,881
- 👥 Sales Rep Performance (8 reps)
- 📦 Brand Performance (AlphaX, BetaPro, DeltaEdge, GammaSuite, OmegaMax)
- 🏭 Industry Performance (7 sectors)
- 🛒 Sales Channel Performance (5 channels)

## Features
- Dynamic filters by Year, Month, and Region
- Interactive bar charts
- PDF export button

## Tools Used
- Tableau
- Microsoft Excel
